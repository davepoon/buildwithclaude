# ThumbGate

Pre-action gates for AI coding agents. Catches destructive operations before they hit your codebase.

- 33 pre-action gates
- Thumbs up/down feedback loop
- Budget enforcement
- NIST/SOC2 compliance tags
- Works with Claude Code, Cursor, Copilot

## Install

```bash
npx thumbgate
```
